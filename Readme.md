# IP Logger

Tool to get victim's ip.

![](https://i.imgur.com/A9Wi56h.png)

## Linux
### Instalation
```
git clone github.com/MomboteQ/IP-Logger
cd IP-Logger
pip3 install -r requirements.txt
```

### Using
```
python3 IP-Logger.py
```

## Windows
### Instalation
```
git clone github.com/MomboteQ/IP-Logger
cd IP-Logger
pip install -r requirements.txt
```

## Using
```
python IP-Logger.py
```
